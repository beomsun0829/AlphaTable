# Let Tab

OpenSource Chrome Extension Tab Manager

![Screenshot 1](./screenshot/screenshot1.png)

## Installation

clone this repo

```
cd package
npm install
npm run build
```

Go to chrome://extensions
Click "Load unpacked"
select "dist" directory
